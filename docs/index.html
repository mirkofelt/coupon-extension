What it does

🏷

Automatic badges

On comparison sites, the code appears right next to the provider — one click copies it.

🔄

Always up to date

Scans all category pages of your source automatically and updates vouchers in the background.

🔍

Searchable

The Settings tab lets you search all stored providers and copy codes directly.

🔒

Local & private

All data stays in the browser. No servers, no tracking requests, no cloud sync.

Installation

CouponAlert is an unpacked extension — load it once, done.

Step 1

Download the ZIP from GitHub and unzip it

Step 2

Chrome: chrome://extensions → enable Developer mode

Step 3

"Load unpacked" → select the extension/ folder

Step 4

Visit your voucher source — the rest runs automatically
