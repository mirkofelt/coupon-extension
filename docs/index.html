CouponAlert erkennt Anbieter auf Vergleichsseiten und zeigt dir gespeicherte Rabattcodes direkt daneben — ohne Suchen, ohne Wechsel.

Download Extension

Popup – Treffer

Settings – Alle Vouchers

Was es kann

🏷

Automatische Badges

Auf Vergleichsseiten erscheint der Code direkt neben dem Anbieter — ein Klick kopiert ihn.

🔄

Immer aktuell

Scannt alle Kategorie-Seiten deiner Quelle automatisch und aktualisiert die Vouchers im Hintergrund.

🔍

Durchsuchbar

Im Settings-Tab kannst du alle gespeicherten Anbieter durchsuchen und Codes direkt kopieren.

🔒

Lokal & privat

Alle Daten bleiben im Browser. Keine Server, keine Tracking-Anfragen, kein Cloud-Sync.

Installation

CouponAlert ist eine unpacked Extension — einmalig laden, fertig.

Schritt 1

ZIP von GitHub herunterladen und entpacken

Schritt 2

Chrome: chrome://extensions → Entwicklermodus an

Schritt 3

„Entpackte Erweiterung laden" → extension/ Ordner wählen

Schritt 4

Auf deiner Voucher-Quelle vorbeischauen — der Rest läuft automatisch
